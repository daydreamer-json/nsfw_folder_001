■テキスト読み上げ機能設定ヘルプ

地の文章や主人公の発言など、音声のない部分に機械的に生成したボイスを割り当てる機能です。
ただし、下記のような制限があります。ご注意ください。

@if exp="!world_object.playerExecMode && typeof global.LineMode=='undefined'"
・メッセージのテキストが最後まで表示され、クリック待ち状態になってから読み上げ機能が実行されます。
@endif
・機械的に読み上げる都合で、間違った読み方で読み上げられる場合があります。
・バックログの音声の無いテキストに対する読み上げ機能はありません。

各種設定はメニューの「[emb exp="GetTextSpeechMenuItemPath()"]」から変更することができます。
@if exp="typeof kag.showAdvancedMenuItem=='Object' && kag.textSpeechMenuItem.parent === kag.troubleShootMenu"
※[emb exp="GetTextSpeechMenuItemPath(kag.troubleShootMenu)"]はメニューの「[emb exp="GetTextSpeechMenuItemPath(kag.showAdvancedMenuItem)"]」にチェックを入れることで表示されます。
@endif

■設定項目の内容説明

設定ダイアログの各項目についての説明です。

・テキスト読み上げ機能を有効にする
　チェックを入れることでこの機能が有効になります。

・音声合成方法
　機械的にボイスを生成する方法を、下記の３種類から選ぶことができます。
　ランタイムが未インストールの状態などで日本語の発声ができない場合は選択できません。

　　* Speech API
　　　　マイクロソフトが開発した音声合成・音声認識用のAPIです。
　　　　Windows XP 以降で使用可能ですが、Windows 8(日本語版)未満の環境では
　　　　日本語の音声合成エンジンは標準で組み込まれておらず、
　　　　製品などを別途購入してインストールすることが必要になります。
　　　　詳細はネット検索など利用して情報を入手してください。

　　　　Windows 8(日本語版)以降では、インストール不要で最初から利用可能になっています。

　　* Microsoft Speech Platform
　　　　マイクロソフトが開発した新しい音声合成・音声認識用のAPIです。
　　　　Windows Vista 以降で利用可能ですが、別途マイクロソフトから
　　　　ランタイムと言語ライブラリのインストールが必要です。（無償で利用できます）

　　　　下記からダウンロードしてインストールすることができます。（英語ページです）
　　　　- ランタイム　　 ： http://www.microsoft.com/en-us/download/details.aspx?id=27225
　　　　　※ 64bit OS環境の場合、64bit版と32bit版ランタイム両方のインストールが必要です
　　　　- 言語ライブラリ ： http://www.microsoft.com/en-us/download/details.aspx?id=27224
　　　　　※ 上記ページ内の MSSpeech_TTS_ja-JP_Haruka.msi をインストールしてください

　　* 外部アプリケーション呼び出し
　　　　指定のコマンドラインアプリケーションを呼び出して、任意のテキスト文章から
　　　　音声合成された wav ファイルを生成する方法です。
　　　　「外部アプリケーション呼び出し」にチェックを入れた後、「選択」ボタンを押して
　　　　コマンドラインアプリケーションを選択してください。
　　　　コマンドラインパラメータの記述方法は、音声パラメータの設定ボタンを押すことで表示されます。

　　　　各アプリごとにコマンドラインの指定が異なるため、動作サポート保障はありません。
　　　　予めご了承ください。

　OSのバージョン別に下記の音声合成方法を推奨します。
　　　　Windows Vista : Microsoft Speech Platform (別途ランタイムのインストールが必要)
　　　　Windows 7     : Microsoft Speech Platform (別途ランタイムのインストールが必要)
　　　　Windows 8以降 : Speech API (そのまま利用可能)

・音声パラメータ
　「声なし発言」と「地の文章」の２種類の音声パラメータを指定できます。
　「声なし発言」は、主人公などの名前表記があるにも関わらずボイスのないメッセージテキストを、
　「地の文章」は、名前表記のないメッセージテキストを指します。

　２種類の音声パラメータにそれぞれ下記の設定から選択できます。

　　* 共通設定
　　　　２種類とも同じ音声パラメータを使用する場合に選択します。
　　　　「共通設定」欄の右端にある「設定」ボタンで音声パラメータを変更できます。

　　* 専用設定
　　　　それぞれ別の音声パラメータを使用する場合に選択します。
　　　　選択欄の右端にある「設定」ボタンで音声パラメータを変更できます。

　　* 無効
　　　　読み上げ機能を無効にします。

　設定ボタンを押すと、「音声合成方法」で指定された形式での設定ダイアログが開きます。
　「Speech API」および「Microsoft Speech Platform」では、話者の選択と速さおよび音程が指定できます。
　「外部アプリケーション呼び出し」では、コマンドラインオプションが指定できます。

　各「発声」ボタンを押すと、「発生テスト文章」に記入されている文章で発声テストができます。

・設定の保存／設定の読み込み
　音声合成方法と音声パラメータの各種設定の保存／読み込みを行います。
　保存した設定は他のゲームでも読み込むことが可能です。

・音量
　生成した音声合成ボイスの音量を指定します。

　　* ゲーム中の全体ボイス音量設定
　　　　ゲーム内の設定値の音量で読み上げます。

　　* 独立した音量設定
　　　　下にあるスライダーの設定値の音量で読み上げます。

・読み上げ辞書
　読み間違いを修正するための辞書を設定できます。
　読み上げテキストに対して、登録されている辞書の置換処理を行った後に、読み上げ処理が実行されます。

　　* 「＋」ボタン
　　　　辞書単語を追加します。
　　　　単語と読みを設定することで、読み上げ前に置換処理を行います。
　　　　なお、単語一覧リストをダブルクリックすると、その単語の読みを編集することができます。

　　* 「↑」「↓」ボタン
　　　　選択されている単語の順序を変更します。
　　　　置換処理の優先順位は最長一致が優先のため、表示順序は特に影響しません。
　　　　単語の分類整理などにご利用ください。

　　* 「＠」ボタン
　　　　単語の全選択／解除を行います。
　　　　なお、単語一覧リストでコントロールキーを押しながら選択すると複数の選択が可能です。

　　* 「×」ボタン
　　　　選択されている単語を削除します。
@if exp="SpeechVoiceSettingDialog.getDefaultDictFile() != ''"
　　　　単語一覧リストが空の状態で押すとデフォルトの辞書に初期化されます。
@endif

　　* インポート
　　　　登録されている辞書単語をファイルに保存します。

　　* エクスポート
　　　　ファイルから辞書単語データを取り込みます。
